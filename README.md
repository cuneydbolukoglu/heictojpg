# HEIC to JPG Converter

Free HEIC to JPG — A lightweight, fast, and macOS-compatible tool that automatically converts HEIC files to JPG.
The app automatically detects the AirDrop folder, monitors incoming files, and converts them instantly with no user action required.

## Features

- Batch convert HEIC files to JPG
- Automatically detects the AirDrop folder (no configuration needed)
- Monitors new AirDrop files in real-time and converts them instantly
- Adjustable output quality (JPG quality level)
- Folder-based scanning and export
- Safe error handling — invalid or unreadable files are skipped
- Fully compatible with macOS
